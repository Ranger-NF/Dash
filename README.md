# How to Play?

### 1. Build Yourself
- Clone this repo
```bash
git clone https://github.com/rohankishore/Dash/
```
- Open the folder and install requirements:
```bash
pip install -r requirements.txt
```

- Run:
```bash
python main.py
```

### 2. Install it via EXE
- Head over to [Releases](https://github.com/rohankishore/Dash/releases/)
- Open the folder
- Run `main.exe`
